# TapTag | Interactive Image Sharing Platform

TapTag is a full-stack social media application that redefines how users interact with visual content. Unlike traditional platforms, TapTag allows creators to define **interactive hotspots** on images, bridging the gap between static media and dynamic user engagement.

## üåü Key Features
- **Interactive Image Editor:** A custom-built workspace for drawing and configuring interactive regions on uploaded images.
- **Advanced Image Processing:** A Python microservice dedicated to image manipulation and steganography.
- **Real-time Engagement:** Live notifications and social feed updates using modern async patterns.
- **Secure Infrastructure:** URL verification via Google Safe Browsing API and optimized media storage with Cloudinary.
- **Financial Integration:** Complete payment flow implementation using Stripe API for creator support.

## üèóÔ∏è Architecture & Tech Stack
The project follows a microservices-inspired architecture to ensure separation of concerns and scalability:

- **Frontend:** React.js, React-Bootstrap, Craco.
- **Primary Backend:** Node.js (Express) with Sequelize ORM.
- **Image Microservice:** Python (Flask), Stegano, Pillow.
- **Database:** MySQL.
- **External Services:** Cloudinary (Media), Stripe (Payments), Google Cloud (Security).
